Big Data é uma área da tecnologia dedicada a examinar grandes volumes de dados para descobrir padrões ocultos.

		---- Data é o novo petróleo ----
A expressão "Data is the new oil." foi utilizada pela primeira vez por um empreeendedor britânico da comercialização de dados chamado Clive HubyEla se tornou famosa quando foi adotada no relatório do Fórum Econômico Mundial chamado Personal Data: The Emergence of a New Asset Class (https://www3.weforum.org/docs/WEF_ITTC_PersonalDataNewAsset_Report_2011.pdf). (2011).

Até 2003, toda a informação digital gerada pelo homem totalizada aproximadamente 5 exabytes. Em 2011, essa mesma quantidade de informação era gerada a cada 2 dias. em 2018, 2.5 quintilhões de bytes eram gerados a cada dia e espera-se que em 2025, essa quantia diária alcance 463 exabytes (SeedScientific - https://seedscientific.com/how-much-data-is-created-every-day/., 2020).

Podemos considerar que o dado é como se fosse o petróleo cru. Apesar de valioso, ele não pode ser utilizado se não for refinado. Assim como o petróleo, se os dados não forem analisados e transformados em informações úteis para a tomada de decisão, seu valor se esvai. Neste momento, surge o grande problema: como podemos processar e analisar um volume de dados tão grande? Acredita-se que 99,5% dos dados produzidos nunca são analisados (Harvard Business Review, 2020).

		---- Big Data ---- 
“Big data is high-volume, high-velocity and high-variety information assets that demand cost-effective, innovative forms of information processing for enhanced insight and decision making.” – Gartner

Essa é a definição oficial data pela consultoria Gartner. O Big Data é um termo "da moda", que engloba a infraestrutura necessária para as empresas minerarem os dados disponíveis para identificar novas oportunidades.

		---- Os 5 Vs do Big Data ----
 Nessa definição, o termo Big Data inclui três dimensões, que se tornaram famosas como os 3 Vs do Big Data: Volume, Velocidade e Variedade.

Com o tempo, essas dimensões foram crescendo, e mais dos "Vs" foram adicionados: Veracidade e Valor.

 * Volume de dados: 	Está relacionado à escala dos dados. Na seção anterior nós já mostramos o volume que é gerado a cada dia pela humanidade. Para se processar e analisar dados, primeiro precisamos ter os dados. Coletá-los, armazená-los e acessá-los sob demanda é um grande desafio do Big Data.
 * Variedade: 	Está relacionado aos tipos de dados obtidos. Antigamente as aplicações só eram capazes de analisar e processar dados estruturados em grandes bancos de dados empresariais. Hoje em dia, é necessário processarmos dados de textos do notícias, blogs, comentários de clientes, avaliações de produtos, etc. Tem-se milhares de dados multimídia, tais como vídeo, imagem e áudio (fala e música). Temos ainda a internet das coisas, que nos provê um oceano de dados de sensores vindo de dispositivos vestíveis, sistemas de monitoramento, geolocalização, sensores e ouros muitos mais. Conseguir extrair estruturas deste lago de dados é outro desafio do Big Data.
 * Velocidade: 	Mas processar variedade e volume não resolve os problemas das empresas. Os dados têm mudado de forma rápida, e a empresa deve ser ágil em identificar tendências para poder agir em tempo. Ou seja, a velocidade dos dados nos apresenta como um grande desafio, uma vez que o dado de hoje já pode ser velho amanhã.
 * Veracidade: 	Quem já ouviu falar em "fake news"? Fontes "confiáveis" e "não confiáveis"? A qualidade dos dados é um dos maiores desafios do Big Data atualmente. O grande volume trouxe dados que muitas vezes não representam um cenário confiável para a tomada de decisão. E a baixa qualidade não se refere apena a dados intencionalmente criados para serem falsos. Ruídos na coleta, incapacidade de se determinar o instante em que o dado foi produzido, entre diversos outros problemas tornam a qualidade dos dados ruim.
 * Valor: 	A informação deve apoiar a tomada de decisão. Mas para os dados se tornarem informação, eles precisam ser analisados por diversas técnicas e ferramentas de Análise de dados (chamada de Analytics). Só assim o dado terá valor. Dado coletado e não armazenado  não vale nada. O Fórum Econômico Mundial, que em 2011 disse que o dado era como o petróleo, publicou em 2018 um post de Adam Schlosser ntitulado You may have heard data is the new oil. It's notLinks para um site externo., em que é afirmado que a cultura de tratar o dado como algo valioso gerou uma ganância nas empresas pela coleta e armazenamento dos dados, como se o dado em si fosse como o ouro. Mas o dado sem a análise não produz valor, e escondê-lo do mundo impede o avanço da sociedade como um todo.
